Opis szczegółowy

Statyczny modul obslugi wejscia (klawiatura + mysz).

Modul bazuje na mapowaniu nazw akcji na fizyczne klawisze/przyciski. Dzieki temu kod gry operuje na akcjach logicznych (np. "Jump").
