Opis szczegółowy

Bazowa jednostka sceny przechowujaca transformacje, komponenty i dzieci.

Klasa implementuje prosty scene graph:

  • pozycja/kat sa lokalne wzgledem rodzica,
  • pozycja/kat w swiecie sa liczone rekurencyjnie.
