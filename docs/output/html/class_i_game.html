Opis szczegółowy

Interfejs bazowy logiki gry uruchamianej przez silnik.

Silnik wywoluje metody tego interfejsu w cyklu zycia aplikacji:

  • Start() raz po inicjalizacji,
  • Update() raz na klatke.
